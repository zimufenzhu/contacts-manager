# 联系人管理系统 - 整合文档

## 项目概述

联系人管理系统是一个基于Kivy和KivyMD的Android联系人管理应用程序，旨在为用户提供在移动设备上管理联系人信息的完整解决方案。

### 功能特性

- 联系人管理（添加、编辑、删除、查看）
- 自定义字段支持
- 联系人搜索功能
- 数据导出（CSV、JSON格式）
- Material Design界面
- 本地SQLite数据库存储

### 技术栈

- **开发框架**: Kivy (Python)
- **UI库**: KivyMD (Material Design组件)
- **数据库**: SQLite
- **打包工具**: Buildozer
- **开发语言**: Python 3.x

## 开发环境搭建

### 环境要求

- Python 3.7+
- Kivy 2.0+
- KivyMD 0.104.1+
- Buildozer (用于Android打包)

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
python main.py
```

## Android打包

使用Buildozer进行Android打包：

1. 安装Buildozer：
   ```bash
   pip install buildozer
   ```

2. 构建APK：
   ```bash
   buildozer android debug
   ```

3. 生成的APK文件位于 `bin/` 目录下

也可以使用提供的批处理文件：
```bash
build_apk.bat
```

## 数据库设计

应用使用SQLite数据库存储联系人信息，包含两个主要表：

1. `contacts` 表：存储联系人基本信息
2. `custom_fields` 表：存储联系人的自定义字段

## 安装说明
1. 下载APK文件到您的Android设备
2. 在文件管理器中找到下载的APK文件
3. 点击APK文件开始安装
4. 如果系统提示"未知来源"，请允许安装来自未知来源的应用
5. 按照屏幕提示完成安装过程

## 使用说明
1. 启动应用后，您将看到主界面
2. 点击"+"按钮添加您的第一个联系人
3. 输入联系人姓名、电话、邮箱等基本信息
4. 点击保存按钮保存联系人
5. 在联系人列表中查看您添加的联系人
6. 点击联系人可以查看详细信息
